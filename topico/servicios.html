FTP Server

FTP (File Transfer Protocol)

Es un protocolo de red utilizado para transferir archivos entre un cliente y un servidor a través de una red TCP/IP, como Internet. Permite subir, descargar, renombrar y eliminar archivos de manera remota, y es ampliamente usado en entornos corporativos, servidores web y administración de sistemas.

Implementacion

Se implementó un servidor FTP gratuito utilizando InfinityFree, un servicio de hosting web que permite subir y gestionar archivos mediante el protocolo FTP. Utilizando la app FTPManager desde un iPhone, se logró establecer una conexión remota con el servidor, cargar archivos en la carpeta pública (htdocs) y acceder a ellos desde cualquier navegador web mediante una URL personalizada. Esta solución demuestra cómo transferir archivos fácilmente entre dispositivos usando una plataforma gratuita y accesible.
